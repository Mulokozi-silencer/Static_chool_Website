Summary of the System....
This project represents a Static school website suitable for a real secondary school.
It includes essential pages and features commonly required by schools, especially in the Tanzanian context.

The system does not use any database and is intentionally kept simple for learning, demonstration, and easy maintenance.

FEATURES
1. Home Page
   School introduction
   Academic Programs
   Photo gallery
   News & Announcement

2. About Us Page
   School overview
   Vision & Mission
   Leadership and Staff section with photos
   Core Values

3. Contact Us Page
   Contact form with Validation
   Sends message via email (no database)

Technologies Used
- Laravel (PHP Framework)
- Blade templates
- HTML5 & CSS3
